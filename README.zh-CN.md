# Claude How To 中文版

这是基于 [luongnv89/claude-howto](https://github.com/luongnv89/claude-howto) 整理的中文学习入口。

当前已经加入一个可直接浏览的中文交互学习页：

- [中文交互学习页](./zh-cn/index.html)

这个页面适合用来快速上手 Claude Code 的核心学习路径，包含：

- 模块切换
- 学习进度保存
- 难度筛选
- 快速开始提示
- 简短自测入口

## 建议怎么用

1. 先打开 [中文交互学习页](./zh-cn/index.html)
2. 从 `Slash Commands` 和 `Memory` 开始
3. 按页面里的 7 天学习节奏继续推进
