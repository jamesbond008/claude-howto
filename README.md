## 这个仓库是做什么的

原始仓库 `luongnv89/claude-howto` 是一套围绕 Claude Code 的系统化学习资料，重点不是单个功能解释，而是把这些能力串成真正可落地的工作流：

- Slash Commands
- Memory
- Skills
- Hooks
- MCP
- Subagents
- Plugins

它更像一条学习路线，而不是一份功能字典。

## 中文学习建议

推荐顺序：

1. Slash Commands
2. Memory
3. Checkpoints
4. CLI Basics
5. Skills
6. Hooks
7. MCP
8. Subagents
9. Advanced Features
10. Plugins

## 直接开始

最快入口：

- 打开 [中文交互学习页](zh-cn/index.html)
- 从 `Slash Commands` 和 `Memory` 开始
- 按页面里的 7 天学习节奏推进
